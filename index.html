What is Full Stack Development?

Full stack development refers to the ability to work on both the front-end (client-side) and back-end (server-side) of web applications. A full stack developer is skilled in designing user interfaces, writing client-side code (HTML, CSS, JavaScript), and building server-side logic and databases. This portfolio demonstrates my understanding of the core technologies used in modern web development.

HTML: The Skeleton of the Web

HTML (HyperText Markup Language) is the standard language for creating web pages. It provides the fundamental structure of sites, which is then enhanced by technologies like CSS and JavaScript. Think of it as the skeleton that gives a website its form.

Commonly Used Tags

HTML uses tags to define content elements. Here are some of the most essential ones:

  • <h1> to <h6> for headings.
  • <p> for paragraphs.
  • <a href="..."> for links (anchors).
  • <img src="..." alt="..."> for images. The `alt` text is crucial for accessibility.
  • <ul> (unordered list) and <ol> (ordered list), with <li> (list item) elements inside.
  • <div> a generic container for grouping content.
Semantic HTML

Using meaningful tags improves accessibility and SEO. These tags explain their content to both the browser and the developer.

<header> <!-- for introductory content or nav links -->
<nav>    <!-- for navigation links -->
<main>   <!-- for the main content of the document -->
<article><!-- for self-contained content -->
<section><!-- for grouping related content -->
<aside>  <!-- for side content (like a sidebar) -->
<footer> <!-- for footer content -->
Forms for User Input

Forms are essential for collecting user data. The <form> element is a container for different types of input elements.

<form>
  <label for="username">Username:</label>
  <input type="text" id="username" name="username">
  
  <label for="password">Password:</label>
  <input type="password" id="password" name="password">
  
  <input type="submit" value="Submit">
</form>

CSS: Styling the Web

CSS (Cascading Style Sheets) is the language used to style and lay out web pages. If HTML is the skeleton, CSS is the clothing—it controls colors, fonts, spacing, and the overall visual presentation.

Selectors and Properties

CSS works by pairing selectors with properties. A selector targets an HTML element, and the properties define its style.

/* Selects all paragraphs */
p {
  color: #333; /* property: value */
  font-size: 16px;
}

/* Selects elements with class="highlight" */
.highlight {
  background-color: yellow;
}

/* Selects the element with id="header" */
#header {
  border-bottom: 2px solid var(--primary);
}
The Box Model

Every element in HTML is a rectangular box. The CSS box model describes how these boxes are sized and spaced. It consists of the content, padding, border, and margin.

Modern Layouts: Flexbox and Grid

CSS has powerful tools for creating complex, responsive layouts.

Flexbox is designed for one-dimensional layouts (a row or a column). It makes it easy to align items vertically and horizontally.

.container {
  display: flex;
  justify-content: space-between; /* Aligns items along the main axis */
  align-items: center;          /* Aligns items along the cross axis */
}

CSS Grid is designed for two-dimensional layouts (rows and columns). It's perfect for creating complex page structures.

.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr; /* Creates three equal columns */
  gap: 1rem;                          /* Adds space between grid items */
}

Bootstrap: Rapid UI Development

Bootstrap is a powerful, open-source CSS framework for responsive, mobile-first web development. It provides a vast library of pre-styled components and a powerful grid system, allowing you to build beautiful websites quickly.

The Grid System

Bootstrap's grid system uses a series of containers, rows, and columns to lay out and align content. It’s built with flexbox and is fully responsive. You use classes like .col-md-6 to specify column widths at different screen sizes (breakpoints).

<div class="container">
  <div class="row">
    <!-- On medium screens and up, this is 6/12 width -->
    <div class="col-md-6">Column 1</div>
    <!-- On medium screens and up, this is 6/12 width -->
    <div class="col-md-6">Column 2</div>
  </div>
</div>
Utility Classes

A core feature of Bootstrap 5 is its extensive library of utility classes. These let you apply styles directly in your HTML without writing custom CSS.

  • Spacing: .m-3 (margin), .p-4 (padding).
  • Colors: .bg-primary, .text-success.
  • Display: .d-flex (display: flex), .d-none (display: none).
  • Borders: .border, .rounded-pill.
<div class="bg-dark text-white p-5 rounded">Hello!</div>
Interactive Components

Bootstrap includes JavaScript plugins for interactive components like modals, carousels, and accordions.

JavaScript: Making the Web Interactive

JavaScript (JS) is a programming language that brings web pages to life. It allows you to create dynamic content, control multimedia, animate images, and much more. It's the engine that powers interactive web experiences.

Variables and Data Types

Variables are containers for storing data. JavaScript has several data types:

  • String: Text, like `"Hello World"`.
  • Number: Numeric values, like `100` or `3.14`.
  • Boolean: Logical values, either `true` or `false`.
  • Object: A collection of key-value pairs, like `{ name: 'Sushank', skill: 'Web Dev' }`.
  • Array: A list of values, like `['HTML', 'CSS', 'JS']`.
const name = "Sushank"; // A string
let age = 25;           // A number
const isDeveloper = true; // A boolean
Functions

Functions are reusable blocks of code that perform a specific task. They are a fundamental building block of JavaScript.

// Defines a function named 'greet' that takes one parameter
function greet(name) {
  return `Hello, ${name}!`;
}

// Calls the function and logs the result
console.log(greet("World")); // Outputs: "Hello, World!"
DOM Manipulation & Events

JavaScript can interact with the HTML and CSS of a page through the Document Object Model (DOM). It can select elements, change their content or style, and react to user actions (events) like clicks, key presses, or mouse movements.

// 1. Select the button element
const myButton = document.getElementById('increment-btn');

// 2. Add an event listener to listen for clicks
myButton.addEventListener('click', function() {
  // 3. This code runs when the button is clicked
  alert('Button was clicked!');
});
